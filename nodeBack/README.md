
# 채팅
### Install
    - 설치
    1. nodeback > npm i
    2. final_project_front > npm i
    

### Setting

    - ip 주소 변경       
    final_project_front > plugin > sokectplugin 에서 자신의 ip주소로 변경

    
### Build

    - build       
    final_project_front > npm run build
    

### Server

    - 서버 열기
    nodeback > node server.js
    실행 후 "sever is listening at localhost:3000" 뜨면 성공
